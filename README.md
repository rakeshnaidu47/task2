
# Task 2 - Phishing Email Analysis

## Objective

The objective of this task was to analyze a phishing email sample and identify common phishing indicators.

## Tools Used

- Sample phishing email
- Web Browser
- GitHub

## Phishing Indicators Identified

- Generic greeting
- Urgent language
- Request for sensitive information
- Suspicious verification link
- Brand impersonation
- Social engineering

## Conclusion

This task helped me understand how phishing emails trick users into revealing sensitive information by creating urgency and pretending to be trusted organizations.
